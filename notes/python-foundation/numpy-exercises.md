### 题目 1：温度数据分析
某城市一周最高气温(摄氏度)为[28,30,29,31,32,30,29]
计算平均气温、最高气温和最低气温
找出气温超过30摄氏度的天数
```
import numpy as np

tem=np.array([28,30,29,31,32,30,29])
print(f"平均气温为{round(tem.mean(),3)}摄氏度") #round函数，精准选择小数位数
print(f"最高气温为{tem.max()}摄氏度") #通过numpy方法实现，更快
print(f"最高气温为{max(tem)}摄氏度")  #通过py内置函数实现，更通用
print(f"最低气温为{tem.min()}摄氏度")

high=tem[tem>30] #布尔索引，筛选满足条件的元素
print(f"气温超过30摄氏度有{high.size}天")
# 平均气温为29.857摄氏度
# 最高气温为32摄氏度
# 最高气温为32摄氏度
# 最低气温为28摄氏度
# 气温超过30摄氏度有2天
```

### 题目 2：学生成绩统计

### 题目 7：统计函数应用
某公司6个月的销售额（万元）为[120,135,110,125,130,140]
计算销售额的总和，均值和方差
找出销售额最高的月份和最低的月份
```bash
prices=np.array([120,135,110,125,130,140])
print(f"销售总和：{prices.sum()}元")
print(f"销售均值：{round(prices.mean(),2)}元")
print(f"销售方差：{round(prices.var(),2)}元")
print(f"销售额最高：{np.argmax(prices)+1}月{prices.max()}元")
print(f"最低：{np.argmin(prices)+1}月{prices.min()}元")

# 销售总和：760元
# 销售均值：126.67元 
# 销售方差：97.22元 
# 销售额最高：6月140元
# 最低：3月110元
```

### 题目 9：综合应用

找出利润最高的天数

cost=np.array([15,18,16,22,20])

print(f"每天的利润：{profit}")
print(f"平均利润：{round(profit.mean(),2)};标准差利润：{round(profit.std(),2)}")
print(f"利润最高是第{bp.argmax(profit)+1}天")
# 每天的利润：[5 7 6 8 8]
# 平均利润：6.8;标准差利润：1.17
